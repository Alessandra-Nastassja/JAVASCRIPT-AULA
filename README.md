## *Como rodar ?* 🚀
### *Tecnologias usadas:*
- [Node](https://nodejs.org/en/);
- Navegador

1. Ter o **[Node js](https://nodejs.org/en/) instalado**;

2. Usando comando abaixo em seu terminal, você poderá rodar:

```

```
